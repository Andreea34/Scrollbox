Scrollbox Demo

Scrollbox is a cross-browser plugin that keeps the full length of your webpage in view. It does this by listening for the window.resize() event, and dynamically adjusting the height of all class .scrollbox elements on the page.

Resize your browser window to see Scrollbox in action.
